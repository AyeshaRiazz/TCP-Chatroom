# TCP Chatroom Application

A simple multi-client chatroom application built using TCP sockets that allows real-time communication between multiple users through a central server.

## Features
- Multi-client support
- Real-time message broadcasting
- Unique user nicknames
- Reliable TCP-based communication

## Technologies Used
- C#
- TCP Sockets
- .NET Console Application

## How It Works
The server listens for incoming client connections and forwards messages between all connected users. Each client can send and receive messages instantly.

## Usage
1. Run the server application first
2. Start multiple client instances
3. Enter a nickname and begin chatting

## Project Type
Console-based client-server networking project
